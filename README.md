## Purchaserequest plugin for GLPI

[![License](https://img.shields.io/badge/License-GNU%20GPL%20v3-blue.svg?style=flat-square)](https://github.com/InfotelGLPI/purchaserequest/blob/master/LICENSE)
[![Web](https://img.shields.io/badge/Web-Infotel-blue.svg?style=flat-square)](https://blogglpi.infotel.com)
[![Translate](https://img.shields.io/badge/Translate-Transifex-cyan)](https://explore.transifex.com/infotelGLPI/GLPI_purchaserequest/)


Plugin extension for Order plugin (https://github.com/pluginsGLPI/order)

Purchaserequest lets users submit purchase requests before launching an order via the Order plugin. Each request goes through a two-level approval workflow: a manually designated approver and an automatic escalation to the General Service Manager when the requested amount exceeds a per-equipment-type threshold.
